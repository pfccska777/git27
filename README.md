Приведите примеры проблем, которые может решить кеширование. 

*Приведите ответ в свободной форме.*

### Решение 1.

Примеры проблем, которые решает кеширование

1. Замедленная загрузка данных
Кеш хранит результаты запросов или вычислений, позволяя быстро получать данные без повторного обращения к медленным источникам (база данных, сеть, диск).

2. Высокая нагрузка на серверы
Повторное извлечение одинаковой информации приводит к большим ресурсозатратам. Кеш снижает количество запросов к серверу или базе, разгружая инфраструктуру.

3. Задержки из-за сетевых запросов
При работе с удалёнными API или сервисами кеширование хранит уже полученные ответы, сокращая время ожидания при повторных обращениях.

4. Частые повторяющиеся вычисления
Сложные операции (например, генерация отчётов, рендеринг страниц) кешируются, чтобы не выполнять их повторно без изменений входных данных.

5. Плохая масштабируемость приложений
Кеш снижает время отклика и позволяет системе обслуживать больше пользователей без значительного увеличения ресурсов.

Кеширование улучшает производительность, экономит вычислительные ресурсы и уменьшает время отклика, делая сервисы более устойчивыми и масштабируемыми.
